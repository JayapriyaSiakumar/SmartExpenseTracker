# Smart Expense Tracker

A React-based expense tracking application designed to help users manage their finances efficiently.

## Features

- Add, edit, and delete expenses
- Categorize expenses
- View expense summaries and analytics
- Track spending over time
- Responsive UI

## Installation

```bash
git clone https://github.com/JayapriyaSiakumar/SmartExpenseTracker.git
cd SmartExpenseTracker
npm install
```

## Usage

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Builds the app for production.

## Project Overview

Smart Expense Tracker is a full-stack React application that simplifies personal finance management. It provides an intuitive interface for tracking daily expenses, categorizing spending, and generating insights into your financial habits.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Quick Start

1. Clone the repository
2. Install dependencies with `npm install`
3. Run `npm start` to launch the development server
4. Open your browser and navigate to the app

## Features in Detail

- **Expense Management**: Quickly add, modify, or remove expense entries
- **Smart Categorization**: Organize expenses by predefined or custom categories
- **Analytics Dashboard**: View visual representations of your spending patterns
- **Historical Tracking**: Monitor spending trends across different time periods
- **Mobile Responsive**: Works seamlessly on desktop and mobile devices
